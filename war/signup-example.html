Example 2: User Signup Form

This example demonstrates:

  1. Processing a form POST request
  2. The POST => Redirect pattern
  3. Pushing validation errors into the model
  4. Exposing a method via a RESTful URL and JSON
  5. Calling server-side code via AJAX
  6. Using JSP as a templating system

Use the form and try:

  • a short username
  • a username of "admin"
  • empty fields
  • a valid form submit

Example 1: The time is...

This is a simple "hello world" type example that shows the use of a model and a view. The model is a representation of the current time (server generated) and examples render this information in different forms:

  1. The time rendered using JSP as a template
  2. The time rendered using Google Closure Templtes (soy) via Silken
  3. The time rendered as JSON
  4. The time rendered as raw text via a response writer

Key files:

  • src/com/googlecode/htmleasy/playground/TheTime.java
  • war/templates/playground/time/timeDisplay.jsp
  • war/templates/playground/time/timeDisplay.soy

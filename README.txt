#Htmleasy Playground

A pre-assembled Eclipse project demonstrating
[Htmleasy](http://code.google.com/p/htmleasy/),
its core features and common patterns.

To run:

 1. Set up Eclipse with the Google App Engine Plugin (GAE)
 2. File -> Import -> Existing Projects into Workspace
 3. Browse and select this directory.
 4. Select the HtmleasyPlayground project at the top level.
 5. Run -> Run As -> Web Application
 6. Open your browser at http://localhost:8888/


Report any issues to: https://github.com/codedance

For more information about Htmleasy visit: http://code.google.com/p/htmleasy/

This code is on github here: https://github.com/codedance/HtmleasyPlayground
